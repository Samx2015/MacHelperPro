Завантажити для Mac

Помічник дисплея в рядку меню Mac

MacHelper Pro

Затемнення Тримає вбудований екран MacBook темним під час дзеркалення на зовнішній монітор.
Керування Дзеркалення, затемнення, сон, яскравість, контраст і скорочення з рядка меню.
Wi-Fi Може перемикатися на сильніший збережений Wi-Fi, коли поточний сигнал слабшає.
Без вікна Dock Без паролів Wi-Fi 14 днів проби
Що це?

Помічник для дзеркалення MacBook.

  • Затемнює вбудований екран затемнює екран MacBook під час дзеркалення на зовнішній монітор
  • Тримає корисне керування поруч дзеркалення, затемнення, сон, яскравість, контраст, скорочення й перемикання Wi-Fi в одному меню
  • Не заважає без вікна Dock, без облікового запису й без аналітики
Чим корисно?

Менше клопоту за столом.

  • Затемнює лише коли потрібно темний при дзеркаленні, нормальний коли екран MacBook є частиною робочого простору
  • Швидкі щоденні дії перемикайте дзеркалення, затемнюйте до натискання клавіші, присипляйте, змінюйте рівні дисплея і дозволяйте Wi-Fi перемикатися за потреби
  • Корисно й тихо працює з рядка меню без вікна Dock, облікового запису або аналітики
Для робочого місця для MacBook, які часто підключені до зовнішнього дисплея
Для швидкого керування меню і є застосунком; немає зайвого вікна
Для приватності без облікового запису, аналітики і паролів Wi-Fi; відгук відкривається як чернетка листа для перевірки
Як отримати

Завантажте і спробуйте 14 днів.

  • Завантажити один застосунок для Apple Silicon та Intel Mac
  • Встановити перетягніть MacHelper Pro до Програм
  • Спробувати усі Pro-функції активні під час проби
macOS 13.0 або новіша arm64 + x86_64 потрібен вбудований екран MacBook
  1. 1 Завантажте DMG.
  2. 2 Перетягніть застосунок до Програм.
  3. 3 Запустіть із Програм.
Підписано розробником і нотаризовано Apple.
